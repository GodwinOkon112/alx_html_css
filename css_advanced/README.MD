##What This Project Does

This project focuses on:

Creating a clean project structure

Writing valid HTML files

Adding custom CSS using an external stylesheet

Using multiple CSS files (base.css and styles.css)

Linking CSS files inside the <head> section of each HTML page

Practising basic layout, typography, and styling

styling a full one page webpage from scratch

##Files Description
index.html

Main page demonstrating basic HTML structure and CSS styling.

base.css

Contains basic styling rules that apply to all pages, such as:

Reset styles

Font rules

Layout defaults

styles.css

Contains specific styling for individual elements and sections.

##How to Link the CSS Files

Inside the <head> section of each HTML file:

<link rel="stylesheet" href="base.css">
<link rel="stylesheet" href="styles.css">


##How to View the Project

Simply open the HTML files in your browser:

Double-click index.html
